# `check_capconnection`

This check returns the status ob the `CapConnection` for the following services:

- workflow-server
- content-feeder
- user-changes
- elastic-worker
- caefeeder-preview
- caefeeder-live
- cae-preview
- studio
- sitemanager
- cae-live



**The Redis service can also be set using the environment variable `REDIS_HOST`.**
